MOBILE PHONE VIEW IN THE WEB SITE PAGE

My case-study is about www.ccmtsl.com Mobile phone device view at www.ccmtsl.com web site page. Firstly, when opening the web site page with your mobile phone device, you will see the navigation bar will be at the top of the web site page and the navigation position is sticky position, meaning the navigation bar can not move to the top, then scrolling down the navigation will can move. Furthermore, you will see the drop-down element at the top left heard corner and it is also a transpierce and the position of the drop-down element is sticky position. Finally, the size of the web site page is small then the tab screen and also the computer device screen and the size of the image, text etc. all of the are small in size more than the other devices.
